v1.14.2 focuses on the first few minutes a new captain spends with Big Bang Smugglers.

This release makes account setup easier to follow, brings invite code entry directly into the mobile app, and aligns the onboarding flow across Android, iOS, and web.

## Smoother First-Time Setup

New players now move through account activation in a cleaner order before they enter the galaxy.

That means fewer confusing handoffs during sign-up and a more predictable path from account creation to getting in-game.

## Invite Codes on Mobile

Invite-only access is no longer something that feels separate from the mobile experience.

If you're joining from Android or iOS, you can now enter your invite code inside the app and continue onboarding from there.

## Better Cross-Platform Consistency

The onboarding flow is now more consistent across every supported platform.

Whether you start on web or mobile, the early account steps now follow the same general path before your captain profile is fully activated.

## Small Presentation Cleanup

This update also cleans up a few rough edges around version display so the current build is shown more consistently inside the app.
