v1.14.3 focuses on the first few minutes a new captain spends with Big Bang Smugglers.

This release makes account setup easier to follow, keeps invite-only access in the web onboarding flow, and leaves the mobile experience more direct.

## Smoother First-Time Setup

New players now move through account activation in a cleaner order before they enter the galaxy.

That means fewer confusing handoffs during sign-up and a more predictable path from account creation to getting in-game.

## Web Invite Access

Invite codes continue to gate the web onboarding flow.

That keeps the web release path intentionally separate while mobile onboarding stays focused on getting verified players into the game quickly.

## Streamlined Mobile Setup

Android and iOS keep the cleaner first-time account flow without inheriting the separate web-only invite step.

That means mobile players can finish setup with less friction while web onboarding still follows its own path.

## Small Presentation Cleanup

This update also cleans up a few rough edges around version display so the current build is shown more consistently inside the app.
