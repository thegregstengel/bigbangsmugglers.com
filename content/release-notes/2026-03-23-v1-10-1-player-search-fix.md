v1.10.1 is a focused usability fix for player search.

This update improves the reliability of searching for players in bounty-related flows and fixes UI friction around the keyboard covering or interfering with the search experience.

## Case-Insensitive Player Search

Player search now works more reliably regardless of capitalization.

This makes search behavior more forgiving and avoids frustrating misses caused by simple case differences in player names.

## Keyboard Overlay Fix

The search flow now behaves better when the on-screen keyboard is active.

This reduces input friction and makes it easier to complete player selection in the UI without the keyboard getting in the way.

## Overall

v1.10.1 is a small but useful polish release that makes player search behave more naturally and keeps the search UI usable while typing.
